# Bucket Rush - README

## Overview

Bucket Rush is a fun and engaging basketball shooting game built using the Phaser framework. Players can choose from different game modes and compete to score as many baskets as possible within the time limit or with limited balls.

## Features

Multiple Game Modes:

Classic Mode: Score as many baskets as you can in 60 seconds.
Time Attack: Score 20 baskets as quickly as possible.
Challenge Mode: Limited to 10 balls; make every shot count!
Customizable UI: The game features a unique 8-bit style with custom fonts and graphics.

Sound Effects: Enjoy immersive sound effects for scoring, misses, and background music.

## Installation

Clone the repository or download the HTML file.
Ensure you have an internet connection to load the Phaser library from the CDN.
Open the HTML file in a web browser to start playing.
## Game Controls

Mouse Click: Aim and shoot the basketball.
Game Buttons: Use the on-screen buttons to navigate through menus and select game modes.
## Assets

Custom images and audio files are required for the game to function properly. Ensure that the following assets are available in the specified directories:
Images: assets/images/
Audio: assets/audio/
